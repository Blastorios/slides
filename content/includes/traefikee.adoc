== As Simple As Traefik

* Install it:

[source,bash]
----
# Cluster Installation
teectl setup --kubernetes --force
teectl setup gen --license="${TRAEFIKEE_LICENSE}" --controllers=1 --proxies=2 | kubectl apply -f -
----

* Configure it:

[source,bash]
----
# Routing Configuration
teectl apply --file=./traefikee/config.toml
----
