== Goal

* We want to host a static webserver behind Traefik

== Problem

* How to tell Traefik to route requests to the web server?

[source,bash,subs="attributes+"]
----
http://{lab-domain}/index.html
  -> Traefik
    -> http://<Webserver Private IP>/index.html
----
